Введение

Клавиатурный тренажер будет использоваться для обучения набору на компьютерной клавиатуре. Он позволяет создавать и выполнять специальные упражнения по набору текста, а также отслеживать статистику их выполнения. Программа использует базу данных PostgreSQL(требует предустановления сервера PostgreSQL и базы данных). Для работа программы требуется доступ в интернет.

Авторизация, запуск программы

Данная программная система состоит из двух частей: для облуживания пользователей и работы администратора. При запуске программной системы появляется окно авторизации, представленное на рисунке. В поля «Логин» и «Пароль» необходимо ввести данные зарегистрированного пользователя и нажать кнопку «Войти». Так же галочка «запомнить меня» позволяет пропустить этап авторизации при следующем запуске системы, если не был произведен выход.

Если данные введены верно, то система определяет права пользователя и происходит вход в систему.

Главное меню

В главном меню есть возможность перейти к списку упражнений или просмотреть статистику.

Список упражнений

После нажатия на упрженение откроется окно с выполнением упражнения.

Просмотр статистики

После выбора в меню вкладки статистика открывается окно с выбором статистики по упражненям, по умолчанию отображается статистика по всем упражнениям.

Выход из системы

После выхода из системы открывается форма авторизации описанная выше.
